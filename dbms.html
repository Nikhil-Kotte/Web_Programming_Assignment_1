Course details

Database Management Systems

Learn relational models, SQL design, transaction management, and modern database internals.

Price: Rs. 1000
Duration: 4 months
Prerequisite: Programming and Data Structures

Course Description

This course covers database design principles, SQL, normalization, indexing, transactions, concurrency control, and recovery mechanisms.

Instructor Details

Course Instructor: Rajan Kapoor

Educational Qualifications: PhD from IIT Bombay, M.Tech from IIT Delhi. Journals Published: 18 international, 11 national.

Course Curriculum

  • Course Objectives
    • Develop strong database design and query skills.
    • Understand transaction processing and optimization.
  • Course Outcomes
    • Create normalized schemas and write efficient SQL queries.
    • Apply concurrency and recovery techniques in DBMS.
  • UNIT-I: Introduction
    • DBMS architecture, ER modeling, schema mapping.
  • UNIT-II: Relational Algebra and SQL
    • Joins, nested queries, aggregation, and views.
  • UNIT-III: Functional Dependency and Normalization
    • 1NF to BCNF, decomposition and dependency preservation.
  • UNIT-IV: Indexing and Transactions
    • B+ trees, hashing, ACID properties and schedules.
  • UNIT-V: Concurrency and Recovery
    • Locking protocols, deadlock handling, logging and checkpoints.
