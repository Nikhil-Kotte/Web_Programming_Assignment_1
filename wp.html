Course details

Web Programming

Develop modern web applications using front-end frameworks and back-end APIs.

Price: Rs. 700
Duration: 4 months
Prerequisite: None

Course Description

This course builds practical skills in front-end and back-end web development using HTML, JavaScript, Vue.js, Node.js, and MongoDB.

Instructor Details

Course Instructor: Raju

Educational Qualifications: PhD from IIT Hyderabad, B.Tech from IIT Kanpur. Journals Published: 10 international, 7 national.

Course Curriculum

  • Course Objectives
    • Learn complete web stack concepts from UI to APIs.
    • Build deployable dynamic web applications.
  • Course Outcomes
    • Create interactive front-end pages and reusable components.
    • Develop REST services and connect with databases.
  • UNIT-I: Web Basics
    • HTTP, browsers, client-server model, web architecture.
  • UNIT-II: HTML5 and XML
    • Semantic tags, forms, multimedia, and structured data.
  • UNIT-III: JavaScript and jQuery
    • DOM manipulation, events, validation and async calls.
  • UNIT-IV: Bootstrap and Vue.js
    • Responsive design, components, directives and state.
  • UNIT-V: Node.js and MongoDB
    • Routing, middleware, CRUD APIs and schema modeling.
